# 🍔 Food Ordering App

A beautiful Flutter-based food ordering app with a modern UI and smooth user experience.


## 🚀 Features
- 🔍 Search food items
- ➕ Add to cart
- 🛒 Cart management
- 💳 Fake payment (GPay UI)
- ✅ Order success screen


## 🛠️ Tech Stack
- Flutter
- Dart


## ▶️ Run the App
```bash
flutter pub get
flutter run -d chrome
